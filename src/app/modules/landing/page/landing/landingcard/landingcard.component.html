vNotes

vNotes atau Vinncent Notes merupakan project pribadi yang saya buat untuk membantu saya dalam memanajemen waktu. Terdapat berbagai macam aplikasi lainnya seperti Notion, etc. tetapi alasan saya tidak menggunakan aplikasi tersebut karena saya merasa kebutuhan saya masih tidak terpenuhi dan oleh sebab itu saya membuat website ini untuk memenuhi kebutuhan saya agar manajemen waktu yang dihasilkan lebih maksimal
